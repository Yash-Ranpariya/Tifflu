**Tifflu** is a high-end Tiffin Service management application designed to bridge the gap between delicious home-cooked meals and busy individuals. Built with a robust **Spring Boot** backend and a sleek **Vanilla JavaScript** frontend, it offers a seamless experience for browsing menus, placing orders, and managing tiffin subscriptions.

---

## ✨ Key Features

- 🥗 **Gourmet Menus**: Browse through a curated selection of healthy and tasty tiffin options.
- 🚀 **Fast Ordering**: Quick and intuitive order placement system.
- 📱 **Responsive Design**: Clean and modern UI that works across all devices.
- 🔐 **Secure Backend**: Powered by Spring Boot for reliability and security.
- 📦 **Order Tracking**: Keep tabs on your daily tiffin deliveries.

## 🚀 Getting Started

Follow these steps to get the project up and running on your local machine.

### 📋 Prerequisites

*   **Java Development Kit (JDK)**: Version 17 or higher.
*   **Maven**: For building the backend.
*   **Node.js**: For running the frontend server.
*   **MySQL**: For local database storage.

---

## 🏗️ Execution Guide

### 1️⃣ Backend (Spring Boot)
The backend service handles the API logic and database interactions.

1.  Navigate to the `backend/` directory.
2.  Configure your database settings in `src/main/resources/application.properties`.
3.  Run the application:
    ```bash
    mvn spring-boot:run
    ```
📍 The API will be available at: `http://localhost:9090`

### 2️⃣ Frontend (Vanilla JS)
The frontend is a lightweight, high-performance web application.

1.  Navigate to the `frontend/` directory.
2.  Install development dependencies:
    ```bash
    npm install
    ```
3.  Launch the application:
    ```bash
    npx serve .
    ```
📍 Access the UI at: `http://localhost:3000`
